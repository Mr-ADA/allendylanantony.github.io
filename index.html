Projects

Online Monitoring of Microservices with Apache Kafka

This is a final project for my bachelor degree in which I have to develop a monitoring of microservices application built on Apache Kafka. In order to monitor the microservices, I implemented an algorithm to get the number of request coming in and calculate the average time of processing the request. Afterwards, I display the result on a line graph and table visualisation on real-time basis.

Tools: KafkaJS, Bootstrap, Node.js, Express.js, MongoDB, Docker, Docker Compose, Microsoft Azure
